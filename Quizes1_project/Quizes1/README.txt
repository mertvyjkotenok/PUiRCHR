Проект Quizes1 (WinForms .NET 8)

Инструкции:
- Откройте решение в Visual Studio 2022/2023: файл Quizes1.csproj
- Файл теста: sample_test.txt (пример). Формат:
  Первая строка - заголовок теста
  Затем блоки вопросов: вопрос строка, далее варианты в формате "Текст;Баллы" пока не пустая строка.
  Разделитель результатов: ---RESULTS---
  Затем строки результатов: Min;Max;Текст

- Положите вашу картинку 123.png рядом с exe (можно заменить в проекте).
- После прохождения теста появится вращающаяся 3D-пирамида. Крутанье мышью — крутит пирамиду.

Файлы:
- Program.cs, MainForm.cs, TestForm.cs, ResultPyramidForm.cs, Quizes1.csproj, 123.png, sample_test.txt
